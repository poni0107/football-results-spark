## Опис задатка

У овом домаћем задатку коришћена је платформа Apache Spark за обраду скупа података о фудбалским утакмицама репрезентација од 1872. године до данас.

Скуп података преузима се са Kaggle платформе:

https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017

У пројекту се користе фајлови:

- `results.csv`
- `goalscorers.csv`

Фајлови са подацима постављени су у `dataset` фолдер пројекта.

## Задатак 1.1 – Најефикасније репрезентације

У првом задатку коришћене су RDD колекције и map-reduce методе.

Потребно је одредити 10 фудбалских репрезентација које су на свим одиграним утакмицама постигле највећи укупан број голова.

У обзир су узети:

- голови постигнути као домаћин (`home_score`)
- голови постигнути као гост (`away_score`)

Резултат је сортиран опадајуће по укупном броју голова.

Колоне резултата:

- `Team`
- `TotalGoals`
